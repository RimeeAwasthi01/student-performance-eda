---

## 📁 Project Overview
This project performs Exploratory Data Analysis (EDA) on student data to understand factors affecting academic performance.

---

## 🎯 Objective
To analyze how study time, attendance, parental support, and extracurricular activities influence student GPA.

---

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

---

## 📊 Key Insights
- Study time positively affects GPA.
- Absenteeism negatively impacts performance.
- Parental support improves student outcomes.
- Balanced extracurricular activities are beneficial.

---

## 📂 Files
- `Student_Performance_Analysis.ipynb` → Full analysis notebook

---

## 🚀 Conclusion
Student performance depends on multiple factors including study habits, attendance, and support systems. Improving these can significantly enhance academic outcomes.
